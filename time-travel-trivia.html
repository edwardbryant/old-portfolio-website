• Time Travel Trivia Quiz •

Thinkful Frontend Developer Project

December 2014
HTML • CSS • SASS • JavaScript • JSON • jQuery • Responsive Design
This was a project completed for Thinkful's Frontend Developer program. The project demonstrates the use of HTML, CSS, JavaScript, JSON, and jQuery to build a simple web-based trivia game. In completing the assignment, I selected a ten question quiz with a theme focused on time travel in popular media (e.g., movies, television, books).
