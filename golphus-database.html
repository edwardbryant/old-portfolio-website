• Golphus Database •

Freelance Developer Project

2011 – 2013
MySQL • Database Design • Redis
This was a long-term freelance project for PressFactory project for their golf handicapping site Golphus.com. I designed a MySQL database for tracking over 18,000 golf courses, users, user games, and related data. I also aided in the creation of backend code for querying and updating the database and provided post-launch database support, including modifying tables, monitoring traffic, and importing data.
