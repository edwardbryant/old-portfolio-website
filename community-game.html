• Community Game •

a.k.a., Journey to the Center of Hawkthorne

2013
Photoshop • Pixel Art • LÖVE engine • Lua
The game is a fan-created retro 2D platformer based on the television show Community. The project was built using the LÖVE game engine. I was an early contributor to the game (credited as Edsterman) who created a large amount of game art, tilesets and levels (including the Greendale campus), and some Lua code. It was a fun project and actress Alison Brie (Community, Mad Men) even tweeted that a level I designed and built (the Dean's closet maze) was her favorite part of the game!
